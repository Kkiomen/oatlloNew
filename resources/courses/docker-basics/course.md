## About this course

This free course teaches you **Docker** from the ground up - no prior experience
needed. You'll learn what containers are, how to run and build your own images, how
to keep data with volumes, connect containers over networks, and run whole
multi-container apps with **Docker Compose**.

Every lesson builds on the previous one, with small, runnable examples you can try as
you go. By the end you'll be able to package a real application (we'll dockerize a
PHP/Laravel app) and follow the practices that keep your images small and safe.

It's written for **developers** who will actually run and build containers day to day,
not just read about them.
